 This program searches SMB/CIFS shared folders of all available machines in the given domain recursively, for files ending with .mp3. Please note that it will only search for files on the intranet. The program is based on Norbert's jCifs client (which is quite different from jCifs)

KNOWN ISSUES:

Will not enumerate more than 500 machines.
Due to obscure reasons, jCifs will not enumerate shares whose names are longer than 12 characters (This behavior is also seen in smbclient).
Program creates one thread per machine, this can bog down your machine!
JDK1.3 is needed for this to work.
